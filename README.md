# Complex-System Project Information-Limited Maxwell Demon

In this repository you can find the work done by my colleague and I about a project of Information-Limited Maxwell Demon.

A guide to the bins are:

- INFORME_DEFINITIVO:
Contains all the final information that is presented in the report, as well as the report itself.

It contains the code used for the project in "code_used/" where you can find the program in C and the plots in two separated notebooks.
The program "PROGRAM.c" is the base and most important. You can change the defined parameters at the beginning and the simulation will change. It exports everythin in a bin "FILES/" and you can see each step of the process. Everything is commented.

The notebook "1_simulation_plots.ipynb" is an old version, containing information of only 1 simulation and the hability to do the video.

The notebook "Stochastic_simulation.ipynb" is THE VERSION containing all the stochastic mean values and deviations as well as the final plots.

- Pruebas:
Contains all the trials and errors each one of us made. This is just in case to have a backup with different systems or configurations.
